Богатыри: Инструкция по программированию

Создаем класс "Богатырь" на Python

Сегодня мы напишем программу, вдохновленную картиной Васнецова "Богатыри". Мы создадим класс "Богатырь" и реализуем его функциональность.

Шаг 1: Определение класса


class Bogatyr:
    def __init__(self, name, weapon, horse_color):
        self.name = name
        self.weapon = weapon
        self.horse_color = horse_color
        self.strength = 100

Шаг 2: Добавление методов


    def defend_rus(self):
        print(f"{self.name} защищает Русь, используя {self.weapon}!")
    
    def ride_horse(self):
        print(f"{self.name} скачет на {self.horse_color} коне.")

Шаг 3: Создание экземпляров


dobrynya = Bogatyr("Добрыня Никитич", "меч", "белый")
ilya = Bogatyr("Илья Муромец", "копье", "черный")
alyosha = Bogatyr("Алёша Попович", "лук", "рыжий")

Шаг 4: Использование методов


dobrynya.defend_rus()
ilya.ride_horse()
alyosha.defend_rus()

Этот код создает трех богатырей и позволяет им защищать Русь и скакать на конях. Попробуйте расширить класс, добавив новые атрибуты или методы, например, метод для описания подвигов богатыря.
